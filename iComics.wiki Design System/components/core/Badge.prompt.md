Compact uppercase mono label for status, source, or counts.

```jsx
<Badge variant="neutral">MangaDex</Badge>
<Badge variant="accent">New</Badge>
<Badge variant="success">Ongoing</Badge>
```

Variants: `neutral` · `accent` · `success` · `danger` · `info` · `solid`. For tappable genre filters use `Tag`; for content ratings use `RatingChip`.
