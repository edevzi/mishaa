Square icon-only button — toolbar actions, reader controls, card overlays. Always pass `label`.

```jsx
<IconButton label="Bookmark" variant="solid"><Bookmark size={18} /></IconButton>
<IconButton label="Settings" variant="ghost-scrim"><Settings size={18} /></IconButton>
<IconButton label="Right-to-left" pressed><MoveLeft size={18} /></IconButton>
```

Variants: `plain` · `solid` · `accent` · `ghost-scrim` (frosted, for over imagery / the reader). `pressed` gives a marigold-tinted toggled look.
